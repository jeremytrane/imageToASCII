# Image to ASCII Converter

The Image to ASCII Converter is a Python application built using the tkinter library for the graphical user interface and the Pillow (PIL) library for image processing. It allows you to convert images into ASCII art, providing a fun and creative way to visualize your favorite pictures.

## Features

- Convert images to ASCII art.
- Adjust the output width to control the level of detail.
- Save the generated ASCII art to a text file.
- User-friendly graphical interface.

## Prerequisites

Before you can use the Image to ASCII Converter, ensure you have the following dependencies installed:

- Python 3
- Pillow (PIL)
- tkinter (usually included with Python)

You can install Pillow using pip:

```bash
pip install Pillow
```
## Usage
1. Clone or download this repository to your local machine.

2. Open a terminal or command prompt and navigate to the project directory.

3. Run the image_to_ascii.py script:
```bash
python image_to_ascii.py
```
1. The graphical user interface (GUI) will appear, allowing you to interact with the application.

2. Follow these steps to convert an image to ASCII art:

a. Click the "Select Image" button to choose an image file (PNG, JPG, JPEG, BMP, or TIFF).

b. Optionally, adjust the "Output Width" to control the width of the ASCII art (higher values provide more detail).

c. Click the "Generate ASCII" button to generate the ASCII art.

3. The generated ASCII art will be displayed in the text area on the GUI.

4. You can save the ASCII art to a text file by clicking the "Save ASCII to File" button.
